VEZZI POS · Prototipo funcional en HTML

Punto de venta web para administrar tu negocio

Esta interfaz está pensada para verse más como un sistema real: acceso por usuario, panel de ventas, productos, clientes, créditos, alertas de stock y reportes rápidos usando tu backend actual.

Venta rápida y venta cliente

Permite cobrar de contado o registrar deuda parcial según el tipo de venta.

Productos y alertas

Búsqueda por nombre o código, edición rápida y alerta visual cuando el stock es bajo.

Clientes y créditos

Consulta de saldo pendiente y registro de pagos parciales desde el mismo panel.

Resumen del día

Ventas totales, utilidad estimada, histórico reciente y alertas clave en un solo lugar.

Importante: tu backend actual no tiene autenticación real con contraseña ni permisos seguros en servidor. Por eso este acceso funciona como demo de rol en frontend: entra con un usuario existente y el panel se adapta a su rol.

Ingresar al sistema

Selecciona un usuario creado en tu backend. Si el rol es Administrador verás módulos completos; si es Cajero verás el flujo de ventas sin acciones de borrado visibles.

Consejo: crea al menos un usuario con rol Administrador y otro con rol Cajero para probar la diferencia visual de permisos.
